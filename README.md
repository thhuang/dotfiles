# dotfiles
My dotfiles!

### [Spacemacs](./spacemacs)
- Install font [Fira Code](https://github.com/tonsky/FiraCode/wiki/Installing)
  ```bash
  $ brew tap homebrew/cask-fonts
  $ brew cask install font-fira-code
  ```
- Install [Ispell](https://en.wikipedia.org/wiki/Ispell) for spell checking
  ```bash
  $ brew install ispell
  ```
